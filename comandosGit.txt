1- Para iniciar repositório
git init
2- para verificar alteracoes no repositorio local
git status
3- Para adicionar todas as alterações do projeto na proxima versao deste
git add .
4- Para gerar uma nova versão
git commit -m "descreva brevemente o que alterou"
(- crie o repositório no git hub
-utilize o comando git remote criado no repositorio remoto para vincular ao repositorio local) 
5- Para enviar as alteracoes para o repositorio remoto
- git push origin main 
6- Para verificar se a versao do repositorio remoto e mais atual que a versao do repositorio 
 git pull
