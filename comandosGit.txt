- para iniciar repositório
git init
- para verificar alteracoes no repositorio local
git status
- para adicionar todas as alterações do projeto na proxima versao deste
git add .
- para gerar uma nova versão
git commit -m "descreva brevemente o que alterou"
- crie o repositório no git hub
-utilize o comando git remote criado no repositorio remoto para vincular ao repositorio local

- para enviar as alteracoes para o repositorio remoto
- git push origin main 

- para verificar se a versao do repositorio remoto e mais atual que a versao do repositorio 
- guit pull
